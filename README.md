# Payment gateway Swahilies

Payment gateway [Swahilies](https://www.swahilies.com/) for PHPNuxBill

## installation

Copy **swahilies.php** to folder **system/paymentgateway/**

Copy **ui/swahilies.tpl** to folder **system/paymentgateway/ui/**
